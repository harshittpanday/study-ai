# 📚 Study AI V1 

A local AI-powered study assistant built using:

- HTML
- CSS 
- JavaScript
- Node.js
- Ollama

## Features

- Local AI chat
- Dark / Light mode
- Chat history
- Editable messages
- Profile customization
- Responsive UI
- Physics & Maths help

## Run Locally

Install dependencies:

npm install

Start backend:

node node.js

Make sure Ollama is installed and running.
